# rules.md

## 絶対ルール
- 指定範囲外を変更しない
- 既存機能を壊さない
- 勝手に機能追加しない
- 不明点は必ず確認する
- 未確定情報を推測で確定しない

## 変更ルール
- 差分変更を基本とする
- 関数単位・モジュール単位で変更する
- 全文再生成は禁止（明示時のみ可）
- テンプレートは汎用性を損なう具体化を避ける

## ファイルルール
- 指定外ファイルは作成しない
- 不要ファイルを増やさない
- 文書テンプレートには記入ルールと未確定時の扱いを含める
- 実プロジェクトで `AGENTS.md` や派生運用文書を作業用に置く場合は、公開要否を判断し、公開不要なら `.gitignore` などで管理対象外にする

## 停止条件
以下の場合は作業停止：
- 範囲不明
- 目的不明
- 環境条件不足
- 設計矛盾
